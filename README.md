# 3D Particle Gesture System

Interactive 3D particle visualization with webcam gesture control built with React and Three.js.

## Features

- 6 different particle patterns (sphere, cube, torus, helix, galaxy, wave)
- Real-time color customization
- Webcam-based gesture control
- Fullscreen mode
- Responsive design

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## Installation

```bash
npm install
```

## Development

Run the development server:

```bash
npm run dev
```

The app will automatically open in your browser at `http://localhost:3000`

## Build

Create a production build:

```bash
npm run build
```

## Preview Production Build

Preview the production build locally:

```bash
npm run preview
```

## Usage

1. Select a particle pattern from the control panel
2. Choose your preferred color
3. Click "Start Camera" to enable gesture control
4. Move your hand closer or farther from the camera to control particle expansion
5. Toggle fullscreen for immersive experience

## Tech Stack

- React 18
- Three.js
- Vite
- Tailwind CSS
